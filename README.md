## Întrebări de control
### Ce face o temă în WordPress și ce face un plugin?
- În WordPress, tema controlează aspectul vizual al site-ului (design, culori, layout), iar pluginurile adaugă funcționalități suplimentare (de exemplu, Contact Form 7 pentru formulare sau Elementor pentru editare vizuală)
### De ce nu se pierde conținutul site-ului atunci când se schimbă tema?
- Conținutul site-ului nu se pierde la schimbarea temei deoarece acesta este stocat în baza de date, iar tema doar îl afișează diferit
### Cum se poate modifica aspectul site-ului fără a edita codul?
- Aspectul site-ului poate fi modificat fără a edita codul folosind secțiunea Appearance → Customize, editorul de blocuri sau un page builder
